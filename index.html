Explain how you use 'fetch' in JavaScript for network requests and how you handle responses?

In JavaScript, we use fetch for network requests, which is a promise-based function. First, we send a request using fetch, then we handle the response using various methods like .then() and .catch() on the returned promise.

What is the difference between map(), filter(), and reduce() in JavaScript, and when do you use each of them?

The functions map(), filter(), and reduce() in JavaScript are used for processing and transforming data. map() is used to change and transform each array element, filter() is used to filter elements based on a condition, and reduce() is used to reduce an array to a single value.

Explain how you use async and await for managing asynchronous operations in JavaScript?

We use async and await to write code that runs asynchronously and manages asynchronous operations better. We use async to define an asynchronous function, and await to wait for a promise to be resolved.

Explain how you add an event listener to a DOM element and how you work with events?

To add an event listener to a DOM element, we use the addEventListener() method. This method allows us to add a function as a listener for a specific event related to the DOM element. Then, we work with the event using the data passed to the listener function.

Explain how you use localStorage and sessionStorage in JavaScript for storing and retrieving local data and what is the difference between them?

We use localStorage and sessionStorage in JavaScript to store local data in the browser. localStorage stores data permanently in the browser, whereas sessionStorage stores data only for the current browser session and removes it after closing the browser window.
